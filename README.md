# NANISH_Game

It is card memory game which combines between Alhasa culture and traditional Saudi music.

# Game Features
1. Combine between enjoyment and strengthen your memory.
2. Challenging vibes.
3. Easy to understand.
4. Tourists Enjoys Alhasa landmarks.
